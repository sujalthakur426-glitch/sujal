# Hyperliquid Sentiment Analysis Project

## Overview
This project analyzes the relationship between market sentiment (Crypto Fear & Greed Index) and trader behavior/performance on the Hyperliquid DEX.

## Setup Instructions
1. Clone the repository: `git clone <your-repo-link>`
2. Ensure the following files are in the root directory:
   - `fear_greed_index.csv`
   - `historical_data.csv`
3. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn
